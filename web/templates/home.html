Todo App

This is an hypermedia-driven todo application built using Golang, HTMX, and SQLite. It provides users with a simple yet efficient interface to manage their tasks. The application is developed primarily for learning purposes in backend development. It utilizes JWT authentication for personal access and performs CRUD operations to interact with the SQLite database.
